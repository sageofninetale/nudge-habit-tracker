NUDGE is a dark-mode, premium-style habit tracking web app.
It helps users build consistent habits with:
- A beautiful overview dashboard
- Daily summary and streaks
- A "Tiny Wins" feed showing recent achievements
- An athlete quotes carousel for motivation
- Clean, mobile-friendly UI

## Tech Stack
- React + Vite
- JavaScript
- LocalStorage for data persistence
- Modern dark UI design with premium aesthetics

## How to Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/sageofninetale/nudge-habit-tracker.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the dev server:
   ```bash
   npm run dev
   ```

4. Open the URL shown in the terminal (typically `http://localhost:5173/`)

## Features

- **Personalized Greeting**: Enter your name on first visit for a personalized experience
- **Habit Tracking**: Add habits with automatic icon assignment
- **Daily Progress**: Visual progress ring showing completion percentage
- **Streak Tracking**: Keep track of consecutive days for each habit
- **Tiny Wins**: Celebrate small victories with a dedicated feed
- **Motivational Quotes**: Rotating carousel of athlete quotes for inspiration
- **Premium Design**: Dark theme with warm accents, smooth animations, and glassmorphism effects
